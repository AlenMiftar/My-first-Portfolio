Jag heter Alen Miftar och är 29 år gammal. Jag bor i Billesholm, som ligger strax utanför Helsingborg. Jag har funnit intresse för kod och programmering sedan sommaren 2023, vilket var startpunkten för min drömresa då jag gick all in för att lära mig koda och programmera. Livsförändringen

En kväll när jag scrollade runt på Instagram dök det upp ett inlägg som förändrade mitt liv!

"Har du ett tråkigt jobb, låg lön, brist på fritid eller ont i kroppen efter jobbet?
Du kan göra något åt det här och nu! Ta reda på om programmering är något för dig helt kostnadsfritt!
Om du klarar av vår introduktionskurs kommer du att få möjlighet att utbilda dig till arbetsfärdig Systemutvecklare. När du sen arbetar som Systemutvecklare kommer du att ha en flexibel och stimulerande vardag, välbetald utan att slita sönder kroppen!
Låter det som en dröm? Nu kan det bli din verklighet! Klicka på länken för att komma till introduktionskursen!"

Jag klickade på länken och fick upp ett formulär där jag fyllde in mina kontaktuppgifter, och fick sedan ett bekräftelse-mail med länk till introkursen av Aladin som själv är systemutvecklare och grundare av företaget "Intensive Code"

I Introkursen fick jag lära mig grunden till Frontend via förinspelade video lektioner om HTML, CSS och JavaScript för att sedan själv kunna bygga en Todo applikation. När det kom till JavaScript med funktioner som onClick, onChange, handleChange för att ta bort en Todo från listan, så fastnade jag och tog kontakt med Aladin. Han förklarade väldigt pedagogiskt så att jag kunde lösa uppgiften, men det stora problemet var att utbildningen är privat och kostar cirka 100.000kr

Drömresan till drömyrket

Jag tog kontakt med SYV och frågade vad som krävs för att läsa på yrkeshögskola till systemutvecklare/programmerare. Jag fick svar att man behöver ha grundläggande behörighet samt ha läst Engelska 6, Matematik 2 och Programmering 1 för att bli behörig till utbildningen Frontend Developer på yrkeshögskolan.

Jag började med att läsa Programmering 1, där jag fick lära mig grunderna i programmeringsspråket C# hos Arena Utbildning. Detta var det första steget jag tog för att kunna uppnå min dröm om att vidareutbilda mig till fullstack-utvecklare.

Jag deltog i Steg till Arbete via Arbetsförmedlingen hos Arcus, där jag fick tips om att Sundsgårdens folkhögskola erbjuder en distanskurs inom Frontend. Efter att ha sökt kursen blev jag antagen och gav järnet för att lära mig så mycket som möjligt och det gjorde jag tack vare lärarna som var super pedagogiska och hjälpsamma!

Jag kan inte tro att det är sant! Efter fem månaders intensiva studier har jag äntligen tagit examen och blivit godkänd i kursen IT - Front end Developer.
Mitt nuvarande delmål är att få chansen att komma ut till företag och bidra med den kunskap jag har samt lära mig så mycket som möjligt. Därefter vill jag avgöra om jag har kapacitet att klara av att jobba som fullstackutvecklare och fortsätta på min drömresa, för att slutligen vidareutbilda mig till fullstackutvecklare!
